# Toutl Theme (VSCode)

The official VSCode port of the Toutl Theme.

> **Grounded. Minimalist. Opaque.**

Toutl is a low-cognitive-load theme for whatever your heart and mind desire. It features earthy tones on a deep matte background, prioritizing content over UI noise.
